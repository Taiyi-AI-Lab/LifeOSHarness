# Media

This folder stores repository media assets used by documentation, demos, and character packs.

Recommended layout:

- `images/`: screenshots, character images, diagrams, and other image assets.
- `videos/`: demo videos, walkthroughs, and other video assets.

Keep large media files out of regular commits when possible. Prefer compressed assets, external release attachments, or Git LFS if the repository begins to carry substantial media.
